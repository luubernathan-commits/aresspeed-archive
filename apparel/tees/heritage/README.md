## Drop Strategy

1. **Soft launch** — HR-08 + HR-09 (Core) as evergreen pieces, available year-round.
2. **Anchor drop** — HR-02 + HR-05 + HR-07 (heritage car trio) as the hero collection. RX-7 + Supra + Civic = the three flagship Ares Speed builds.
3. **Limited release** — HR-03, HR-04, HR-06 as numbered Limited (≤100 units each), hang-tag stamped with serial number.
4. **Editorial pieces** — HR-01 + HR-10 carry the brand-story narrative. Use for lookbook editorial, magazine partnerships, and SAU/SS forum nostalgia plays.

## Universal Negative Prompt Block

```
no cheesy chrome gradients, no AI-deformed hands or fingers, no generic tuner clichés,
no modern 2020s car designs, no Comic Sans or default fonts, no glossy bevels,
no watermarks, no Fast & Furious logos or trademarked IP, no Nissan Skyline / GT-R
unless owned by Ares Speed, no fake or decorative kanji, no stock-photo backgrounds,
no white tees unless specified, no AI smudging on linework, no extra fingers,
no warped wheels, no melted typography
```

## Per-SKU Folder Layout

Each `HR-XX-*/` folder is scaffolded for the following deliverables:

```
HR-XX-name/
├── prompts.md           ← (a) Nano Banana / (b) Canva / (c) Midjourney v7 / (d) Negative
├── artwork-flat.png     ← transparent PNG, 300 DPI (final art)
├── mockup-front.jpg     ← AS Colour 5026 Heavy Tee front mockup
├── mockup-back.jpg      ← back mockup (where applicable)
└── lifestyle.jpg        ← model + tee + heritage car, sodium-light Brisbane night
```

`prompts.md` is committed in this initial scaffold. Image deliverables to be generated and committed in subsequent passes.
